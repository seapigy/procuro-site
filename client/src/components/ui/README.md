# shadcn/ui Components

This directory is for shadcn/ui components.

## Setup Instructions

To set up shadcn/ui in this project:

1. Install dependencies:
```bash
npm install class-variance-authority clsx tailwind-merge
npm install -D @radix-ui/react-slot
```

2. Initialize shadcn/ui:
```bash
npx shadcn-ui@latest init
```

3. Add components as needed:
```bash
npx shadcn-ui@latest add button
npx shadcn-ui@latest add card
npx shadcn-ui@latest add input
```

## Configuration

The `tailwind.config.js` and `tsconfig.json` files are already configured for shadcn/ui compatibility.
